# Приложения для обучения

## Описание

Приложение позволяет организовывать обучение учеников в группе

## Функциональность

Приложение предоставляет следующие возможности:

- Выдача домашних заданий;
- Проведение онлайн занятий;
- Организация школьного пространства;
- Контроль обучения;
